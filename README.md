# Utolsó pálcika

Egyszerű pálcikás játék. A játék lényege, hogy a megadott pálcikákból
felválltva vesznek el a játékosok. Dönthetnek, hogy egyet vagy kettőt vesznek el.
Az nyer, aki elveszi az utolsót.
